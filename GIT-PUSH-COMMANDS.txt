TROUBLESHOOTING:
================

If you get "fatal: Unable to create '.git/index.lock'":
1. Close any text editors or IDEs with the repository open
2. Run: git status
3. If that fails, close all applications and retry

If git push fails with "permission denied":
1. Verify you have push access to https://github.com/CINTENT-LAB/api-cintent
2. Check GitHub SSH key or personal access token is configured

If the commit message is too long:
1. Use the shorter version:
   git commit -m "CINTENT Platform v2 - Foundation Complete. Metadata-driven architecture, 8 enterprise systems, frozen schema v1.0. Ready for stabilization phase."
