# Travel Domain API Contracts

All Travel APIs are domain-generic:

- `POST /api/domain/travel/itinerary`
- `POST /api/domain/travel/continuity`
- `POST /api/domain/travel/replay`
- `POST /api/domain/travel/telemetry`
- `POST /api/domain/travel/emergency`
- `POST /api/domain/travel/disruption`
- `POST /api/domain/travel/traveler-state`

No application-specific route names are permitted in this domain pack.
